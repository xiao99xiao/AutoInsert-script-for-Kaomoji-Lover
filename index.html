What's this for

Pasting something like a Kaomoji from any sandboxed Mac apps requires an Apple Script. According to Apple's requirement, no app is allowed to install this Apple Script automatically. Instead, the installation has to be performed by the user (which is you) manually. So to make things easier, I've made this dmg for you.

How to Install

Download and open the DMG here, drag the 'paste.scpt' to 'Drop here' icon on the right side. Done.
