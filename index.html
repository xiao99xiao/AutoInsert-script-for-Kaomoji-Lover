Autoinsert script for Kaomoji Lover

An Apple Script helps inserting Kaomojis to current input.

View project on GitHub

What's this for

Pasting something like a Kaomoji from any sandboxed Mac apps requires an Apple Script. According to Apple's requirement, no app is allowed to install this Apple Script automatically. Instead, the installation has to be performed by the user (which is you) manually. So to make things easier, I've made this dmg for you.

How to Install

  1. Download and open the DMG here.
  2. Drag the 'Drop here' icon to desktop.
  3. Drag the 'paste.scpt' to 'Drop here' icon on the desktop. Done.

このファイルとは

顔文字 for Mac の顔文字自動入力機能はある Apple Script に依存しています。この Apple Script をインストールために、DMG ファイルを用意しました。アプリの Script インストール機能は、Apple の要求により、撤回しました。

インストール

  1. ここをクリックして、DMG ファイルをダウンロードして開けて。
  2. 右の'Drop here'をデスクトップへ移動。
  3. 左の'paste.scpt'をデスクトップの'Drop here'にドラッグ。以上。

这是什么

颜文字 for Mac 的自动插入颜文字功能需要用户（也就是您）手动安装一个 Apple Script 文件。为了简化操作，我们准备了一个 DMG 文件给你。之前 App 提供的自动安装 Script 功能，应 Apple 的要求，将会移除。

如何安装

点此下载并打开 DMG 文件，先将右侧的 'Drop here' 拖动到桌面上，然后将左侧的 'paste.scpt' 拖到桌面上的 'Drop here' 上即可。
