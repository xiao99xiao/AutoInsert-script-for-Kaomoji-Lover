Autoinsert script for Kaomoji Lover

An Apple Script helps inserting Kaomojis to current input.

View project on GitHub

What's this for

Pasting something like a Kaomoji from any sandboxed Mac apps requires an Apple Script. According to Apple's requirement, no app is allowed to install this Apple Script automatically. Instead, the installation has to be performed by the user (which is you) manually. So to make things easier, I've made this Automator for you.

How to Install

  1. Download and open the Zip file.
  2. Run the EnableAutoInsert.app once.Done.

顔文字 for Mac の顔文字自動入力機能はある Apple Script に依存しています。この Apple Script をインストールために、Automator を用意しました。アプリの Script インストール機能は、Apple の要求により、撤回しました。

这是什么

颜文字 for Mac 的自动插入颜文字功能需要用户（也就是您）手动安装一个 Apple Script 文件。为了简化操作，我准备了一个 Automator 小程序。之前 App 提供的自动安装 Script 功能，应 Apple 的要求，将会移除。

如何安装

  1. 点击右侧的 Zip 文件下载链接，然后解压。
